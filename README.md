# EMBA Study Pro 🎓

Công cụ ôn thi đầu vào chương trình Thạc sĩ điều hành cao cấp EMBA — Đại học Ngoại thương.

## Tính năng
- 🃏 **Flashcard** — 50+ thuật ngữ Quản trị học từ Robbins & Coulter
- 🤖 **Phỏng vấn AI** — Claude AI chấm điểm câu trả lời theo chuẩn EMBA
- 📝 **Trắc nghiệm** — 10 câu hỏi có giải thích đáp án chi tiết

## Cài đặt và chạy

```bash
npm install
npm run dev
```

Mở trình duyệt tại `http://localhost:5173`

## Deploy

Project này đã được cấu hình sẵn cho Railway.app.
Chỉ cần push lên GitHub và connect với Railway là xong.
